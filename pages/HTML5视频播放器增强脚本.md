- ## HTML5视频播放器增强脚本-快捷键列表
  
  | 快捷键 | 说明 |
  | --- | --- |
  | ctrl+\\ | 快捷键是否全网页可用，默认true |
  | Ctrl+space | 禁用/启用 该播放插件 |
  | Enter | 进入全屏 |
  | shift+Enter | 进入网页全屏 |
  | shift+P | 进入或退画中画功能 |
  | shift+T | 截图，截取当前画面并保存 |
  | N | 下一个/集视频（仅部分网站支持） |
  | i | 上一帧 (截图时进行微调以找到质量最佳的一帧) |
  | k | 下一帧 (不支持netflix，因为快捷键冲突) |
  | y| 亮度增加% |
  | p | 亮度减少% |
  | t | 对比度增加% |
  | e | 对比度减少% |
  | Q | 图像复位 |
  | S | 画面旋转 90 度 |
  | shift+C | 放大视频画面 +0.1 |
  | shift+X | 缩小视频画面 -0.1 |
  | shift+Z | 恢复视频画面 |
  | shift+R | 启用或禁止自动恢复播放进度功能 |
  | shift+→ | 画面向右移动10px |
  | shift+← | 画面向左移动10px |
  | shift+↑ | 画面向上移动10px |
  | shift+↓ | 画面向下移动10px |
